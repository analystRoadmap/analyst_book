# IA/LLM

Интеллектуальные ассистенты и большие мультимодальные модели становятся частью повседневной работы аналитика. Они ускоряют поиск информации, помогают проверять требования и формировать документацию на основе корпоративных знаний.

В этом разделе собраны направления, где стоит углубиться, чтобы уверенно использовать современные модели в продуктах и процессах.
